# 🏥 Automated Healthcare Diagnostic System using AutoML

## 🚀 Overview
This project is an **AI-powered Healthcare Dashboard** that centralizes patient data, medical reports, and hospital services into a single platform. It helps patients and doctors access, analyze, and understand medical information efficiently.

---

## 🎯 Problem Statement
Modern healthcare systems store data in fragmented systems, making it difficult for patients and doctors to access medical records, understand reports, and make quick decisions. This leads to inefficiency, poor communication, and delays in treatment.

---

## 💡 Solution
We developed an **Integrated Patient Care Dashboard** that uses AI and Machine Learning to:
- Analyze medical reports
- Predict health risks (e.g., diabetes)
- Generate simple summaries of complex reports
- Provide real-time access to healthcare data

---

## ✨ Features
- 📊 Centralized Patient Dashboard
- 🤖 AI-based Report Analysis
- 📑 Medical Report Summarization
- ⚡ Risk Prediction using ML (XGBoost)
- 🔍 Keyword Extraction (TF-IDF)
- 👨‍⚕️ Doctor & Patient Role-based Access
- 🏥 Hospital Navigation Support

---

---

## ⚙️ How It Works
1. User uploads medical report  
2. System stores data securely  
3. AI processes the report  
4. Generates:
   - Key medical terms  
   - Risk prediction  
   - Simplified summary  
5. Results are shown on dashboard
